# Nuxt Setup

## Steps

```bash
# First of all create a new .env file and add backend url
$ touch .env

# add the below line in your .env file
$ BASE_URL=http://127.0.0.1:8000

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate
```

```bash
# The implementation can be found on the url:
$ http://localhost:3000
```
